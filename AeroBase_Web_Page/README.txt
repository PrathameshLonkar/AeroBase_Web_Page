Steps for execution

Note: install packages/dependencies specified in package-lock.json


open terminal in final_aerobase

node mysql.js (this will execute the script on localhost on 3001)

open another terminal in final_aerobase

npm start



This will start the development server on web browser on localhost 3000


Note: wait for couple of minutes for data to load as it is a huge database

Snapshots of web application is attached in the snaps folder.
